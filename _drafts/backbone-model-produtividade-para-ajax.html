Backbone Model

Você não precisa gastar horas estudando toda a documentação do Backbone para que consiga ser produtivo com ele. Neste post vou lhe mostrar como tirar proveito dos recursos AJAX do Backbone Model para simplificar seu código e diminuir significantemente o tempo gasto codificando suas requisições AJAX.

Backbone JS

Um dos grandes diferencias deste framework é a simplicidade e eficácia em prover soluções para para grandes sistemas ou pequenos sites. Isto é possível devido ao fato de que foram adotados padrões de projetos simples e objetivos na sua construção, como por exemplo o Active Record utilizado nos Models. Isso significa que o Model que você cria com o Backbone é totalmente responsável por suas funcionalidades, não será preciso criar uma infinidade de arquivos e classes para fazer o trabalho do seu Model, como por exemplo, validar, salvar excluir, executar as requisições AJAX dentre outros.
